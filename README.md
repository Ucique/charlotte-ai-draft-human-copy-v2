# Charlotte — AI Draft → Human Copy (DACH) (v2)

Static landing page (DE/EN) for “Charlotte — AI Draft → Human Copy (DACH)”.

- No build tools
- No framework
- GitHub Pages ready
- Language toggle (DE/EN) + localStorage
- Contact form opens mail app via mailto (no backend)
- Modals via CSS :target + ESC close
- Respects prefers-reduced-motion

## Files

- index.html
- styles.css
- app.js
- README.md

## Run locally

Just open `index.html` in a browser.
